Description

Akin to Google's Code Search, this aims to search all the code found on CPAN. It concenrates on searching code; if you just want to look for a module use search.cpan.org instead.

Regular Expressions are supported via re::engine::RE2 (which wraps RE2 that originated from Google's Code Search). This means that the regexps supported are mostly Perl compatible but are not totally compatible with Perl. Of particular note is look-ahead and look-behind are not supported, for more details see RE2's supported syntax. A future version may implement more advanced syntax so it is not just limited to regexps, and can filter out matches by NOT, but this isn't yet supported.

(Due to performance concerns re::engine::RE2's ability to fallback to Perl regexps is disabled.)
